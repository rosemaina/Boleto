# Boleto

### To use this repo:

##### Clone the repository
`https://github.com/rosemaina/Boleto.git`

##### Create a workspace
* To create a Podfile - `pod init`
* Add this pods:
`pod 'Firebase/Analytics'`
`pod 'Firebase/Auth'`
`pod 'Firebase/Firestore'`
 `pod 'Firebase/Database'`
  `pod 'Firebase/Core'`
  `pod 'Firebase/Storage'`
  `pod 'FLAnimatedImage'`
  `pod 'GoogleSignIn'`
  `pod 'ImagePicker'`
 `pod 'IDMPhotoBrowser'`
  `pod 'Material'`
  `pod 'MBProgressHUD'`
  `pod 'ProgressHUD'`
  `pod 'TTTAttributedLabel'`
  `pod 'lottie-ios'`
  
  * Run - `pod install`
  * Open `open Boleto.xcworkspace`
